# Chrome Web Store - 스크린샷 가이드

## 필요한 스크린샷 (최대 5개)

### 1. 메인 팝업 화면 (라이트 모드)
**크기**: 1280x800 또는 640x400  
**내용**:
- 홈 탭이 열린 상태
- 오늘 총 사용 시간 표시
- 시간 제한 설정 섹션
- 네비게이션 바 (홈, 대시보드, 사이트, 설정)
- Material Design UI

**캡처 방법**:
1. 확장 프로그램 팝업 열기
2. 라이트 모드로 설정
3. 홈 탭 선택
4. 스크린샷 캡처 (1280x800 권장)

### 2. 다크 모드 화면
**크기**: 1280x800 또는 640x400  
**내용**:
- 다크 모드로 전환된 상태
- 동일한 기능이 다크 테마로 표시
- 눈의 피로를 줄이는 어두운 배경

**캡처 방법**:
1. 테마 토글 버튼 클릭하여 다크 모드로 전환
2. 홈 또는 대시보드 탭 선택
3. 스크린샷 캡처

### 3. 대시보드 화면
**크기**: 1280x800 또는 640x400  
**내용**:
- 대시보드 탭 선택
- 일간 사용량 (시간대별) 차트
- 주간 사용량 (요일별) 차트
- 월간 사용량 (주차별) 차트
- 상위 사이트 리스트
- 전체 통계 섹션

**캡처 방법**:
1. 대시보드 탭 클릭
2. 모든 아코디언 섹션 열기
3. 스크린샷 캡처

### 4. 사이트 리스트 화면
**크기**: 1280x800 또는 640x400  
**내용**:
- 사이트 탭 선택
- 사이트별 사용 시간 리스트
- 각 사이트의 파비콘과 사용 시간 표시
- 정렬된 사이트 목록

**캡처 방법**:
1. 사이트 탭 클릭
2. 여러 사이트가 표시되도록 확인
3. 스크린샷 캡처

### 5. 설정 화면
**크기**: 1280x800 또는 640x400  
**내용**:
- 설정 탭 선택
- 자동 추적 토글
- 알림 토글
- 언어 선택 드롭다운
- 데이터 내보내기 버튼
- 데이터 초기화 버튼

**캡처 방법**:
1. 설정 탭 클릭
2. 모든 설정 옵션이 보이도록 확인
3. 스크린샷 캡처

## 스크린샷 요구사항

### 형식
- **JPEG** 또는 **24비트 PNG** (알파 채널 없음)
- **크기**: 1280x800 또는 640x400 픽셀
- **최대 5개**

### 품질 가이드라인
- 선명하고 깨끗한 이미지
- 텍스트가 읽기 쉬워야 함
- 중요한 기능이 명확히 보여야 함
- 브랜드 일관성 유지

### 제외해야 할 내용
- 개인정보 (URL, 검색어 등)
- 민감한 데이터
- 저작권이 있는 콘텐츠

## 스크린샷 캡처 팁

1. **Chrome 확장 프로그램 팝업 크기 조정**
   - 팝업을 열고 개발자 도구로 크기 확인
   - 필요시 CSS로 크기 조정

2. **고해상도 캡처**
   - 브라우저 확대/축소를 100%로 설정
   - 고해상도 모니터 사용 권장

3. **일관된 스타일**
   - 모든 스크린샷에서 동일한 테마 사용
   - 동일한 데이터 세트 사용

4. **편집 도구**
   - 필요시 이미지 편집 도구로 크기 조정
   - 텍스트 오버레이 추가 가능 (선택사항)
