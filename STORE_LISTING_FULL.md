## 📝 제품 세부정보

### 패키지 제목
```
Tab Timer
```

### 패키지 요약 (132자 이내)
```
Track your browser time and boost productivity. See how much time you spend on each website and improve your focus.
```

### 설명 (16,000자 이내)

**Tab Timer**는 여러분의 온라인 시간을 더 가치있게 만들어주는 스마트한 시간 추적 도구입니다. 웹사이트별 사용 시간을 자동으로 추적하고 분석하여 생산성을 높이고 더 나은 디지털 습관을 형성할 수 있도록 도와줍니다.

#### 🎯 주요 기능

**자동 시간 추적**
- 웹사이트 방문 시 자동으로 시간 기록
- 세션별 상세 추적으로 정확한 사용 패턴 파악
- 백그라운드에서 조용히 작동하여 사용자 경험을 방해하지 않음
- 실시간으로 사용 시간 업데이트

**실시간 통계 & 대시보드**
- 오늘의 총 사용 시간을 한눈에 확인
- 일간, 주간, 월간, 년간 사용량 통계 제공
- 시간대별, 요일별, 주차별 사용 패턴 분석
- 상위 사이트 및 전체 통계 제공
- 아코디언 형식의 직관적인 대시보드

**시간 제한 관리**
- 원하는 사이트에 시간 제한 설정
- 설정 시간 초과 시 부드러운 알림
- 실시간 남은 시간 표시
- 시간 제한 초과 시 자동 알림
- 더 건강한 인터넷 사용 습관 형성

**사이트별 상세 분석**
- 각 사이트의 사용 시간 상세 확인
- 사이트별 시간대별 사용 패턴
- 전체 대비 사용 비율 표시
- 일일 평균 사용 시간 확인

**아름다운 인터페이스**
- Material Design 기반의 모던한 UI
- 다크 모드 지원으로 눈의 피로 감소
- 부드러운 애니메이션과 직관적인 조작
- 반응형 디자인으로 모든 화면 크기 지원
- 탭 기반 네비게이션으로 쉬운 접근

**다국어 지원**
- 한국어 및 영어 지원
- 언어 설정을 통한 쉬운 전환
- 모든 UI 요소의 완전한 현지화

**개인정보 보호 최우선**
- 모든 데이터는 로컬에만 저장
- 외부 서버로 전송하지 않음
- 오프라인에서도 정상 작동
- 데이터 내보내기 및 초기화 기능 제공

**데이터 관리**
- JSON 형식으로 데이터 내보내기
- 데이터 초기화 기능
- 자동 데이터 정리 (10MB 초과 시)
- 장기간 데이터 보관 지원

#### 💡 이런 분들께 추천합니다

- 시간이 어떻게 흘러가는지 궁금하신 분
- 더 생산적인 하루를 보내고 싶으신 분
- SNS나 유튜브 사용을 줄이고 싶으신 분
- 건강한 디지털 라이프를 원하시는 분
- 웹 사용 습관을 개선하고 싶으신 분
- 시간 관리가 필요한 학생 및 직장인

#### 🚀 시작하기

1. **설치**: Chrome 웹 스토어에서 Tab Timer를 설치하세요
2. **자동 추적**: 설치 후 자동으로 웹사이트 사용 시간이 추적됩니다
3. **통계 확인**: 브라우저 툴바의 Tab Timer 아이콘을 클릭하여 오늘의 사용 통계를 확인하세요
4. **시간 제한 설정**: 필요시 특정 사이트에 시간 제한을 설정하세요
5. **대시보드 확인**: 상세한 통계와 인사이트를 대시보드에서 확인하세요

#### 🔒 개인정보 보호

Tab Timer는 사용자의 개인정보 보호를 최우선으로 생각합니다:

- **로컬 저장**: 모든 데이터는 Chrome의 로컬 저장소에만 저장됩니다
- **서버 전송 없음**: 데이터는 외부 서버로 전송되지 않습니다
- **도메인만 수집**: 전체 URL이 아닌 도메인 이름만 수집합니다
- **사용자 제어**: 언제든지 데이터를 삭제하거나 내보낼 수 있습니다

자세한 내용은 개인정보 처리방침을 참조하세요: https://octxxiii.github.io/Tab-Timer-Pages/

#### 🛠️ 기술 사양

- **플랫폼**: Chrome Extension (Manifest V3)
- **최소 Chrome 버전**: 88
- **저장 공간**: 최대 10MB (자동 정리 기능 포함)
- **오프라인 지원**: 네트워크 연결 없이도 작동
- **권한**: storage, tabs, activeTab, notifications
