### 📝 변경 사항

#### Added
- `src/` 디렉터리 구조
- `public/` 정적 파일 디렉터리
- `dist/` 빌드 출력 디렉터리
- `scripts/build.js` 빌드 스크립트
- `src/utils/` 공통 유틸리티 모듈
- `PROJECT_STRUCTURE.md` 프로젝트 구조 문서
- 에러 핸들링 및 복구 로직

#### Changed
- 파일 구조 재구성
- 빌드 프로세스 개선
- 코드 모듈화
- 성능 최적화 (디바운싱)

#### Fixed
- Git merge conflict 해결
- 문법 오류 수정
- 메모리 누수 방지
- 데이터 저장 안정성 개선

#### Removed
- 중복 코드 제거
- 사용하지 않는 파일 정리 (tab.js, popup.css)

### 🔧 기술적 개선

- **에러 핸들링**: try-catch 블록 추가, 안전한 폴백
- **성능**: 디바운싱된 저장, 효율적인 메모리 관리
- **안정성**: 데이터 검증, 상태 복구 로직
- **유지보수성**: 모듈화, 명확한 구조

### 📦 빌드 및 배포

- `npm run build`: 프로덕션 빌드
- `npm run clean`: 빌드 출력 정리
- `dist/` 디렉터리를 Chrome 확장 프로그램으로 로드
