# Chrome Web Store - 프로모션 타일 가이드

## 작은 프로모션 타일 (Small Tile)

**크기**: 440x280 픽셀  
**형식**: JPEG 또는 24비트 PNG (알파 없음)

### 디자인 요소
- Tab Timer 로고/아이콘 (왼쪽 또는 중앙)
- "Tab Timer" 텍스트
- 간단한 슬로건: "시간을 추적하고 생산성을 높이세요"
- 배경: 그라데이션이나 단색 (Material Design 색상)

### 색상 팔레트
- Primary: #2196F3 (파란색)
- Background: #FFFFFF 또는 #F5F5F5
- Text: #212121

## 마키 프로모션 타일 (Marquee Tile)

**크기**: 1400x560 픽셀  
**형식**: JPEG 또는 24비트 PNG (알파 없음)

### 디자인 요소
- Tab Timer 로고 (왼쪽)
- 큰 제목: "Tab Timer"
- 부제목: "웹 사용 시간 추적 및 생산성 향상"
- 주요 기능 아이콘 또는 일러스트레이션
- 배경: 그라데이션이나 패턴

### 레이아웃 제안
```
[로고] Tab Timer
       웹 사용 시간 추적 및 생산성 향상
       [주요 기능 아이콘들]
```

## 디자인 가이드라인

### 텍스트
- 명확하고 읽기 쉬운 폰트 사용
- 최소 12pt 크기
- 높은 대비 (텍스트와 배경)

### 이미지
- 고해상도 (최소 요구 크기의 2배 권장)
- 선명하고 깨끗한 이미지
- 브랜드 일관성 유지

### 색상
- Material Design 색상 팔레트 사용
- 브랜드 색상 (#2196F3) 활용
- 접근성을 고려한 색상 대비

## 제작 방법

### 옵션 1: 디자인 도구 사용
- Figma, Adobe XD, Canva 등 사용
- 템플릿 크기로 새 문서 생성
- 디자인 요소 배치
- PNG/JPEG로 내보내기

### 옵션 2: HTML/CSS로 생성
- HTML 파일로 디자인 생성
- 브라우저에서 렌더링
- 스크린샷 캡처 또는 HTML to Image 변환

### 옵션 3: 이미지 편집 도구
- Photoshop, GIMP 등 사용
- 요구 크기로 새 캔버스 생성
- 디자인 요소 추가
- 최종 이미지 내보내기

## 참고 사항

- 프로모션 타일은 선택사항이지만 권장됨
- 스토어에서 더 눈에 띄게 표시됨
- 브랜드 인지도 향상에 도움
- 사용자에게 첫인상을 제공
