## 🚀 제출 프로세스

### 1. Chrome 웹 스토어 개발자 계정
1. [Chrome Web Store Developer Dashboard](https://chrome.google.com/webstore/devconsole) 접속
2. Google 계정으로 로그인
3. 개발자 등록비 $5 결제

### 2. 새 항목 추가
1. "새 항목" 클릭
2. tab-timer.zip 업로드
3. 스토어 목록 정보 입력
