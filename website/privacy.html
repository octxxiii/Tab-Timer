Tab Timer 개인정보 처리방침

최종 업데이트: 2024년 1월

Tab Timer는 사용자의 개인정보를 중요하게 생각합니다.

이 확장 프로그램은 모든 데이터를 사용자의 로컬 기기에만 저장하며, 외부 서버로 전송하지 않습니다.

1. 수집하는 정보

Tab Timer는 다음과 같은 정보를 수집합니다:

  • 웹사이트 방문 기록: 방문한 웹사이트의 도메인 이름과 체류 시간
  • 사용 통계: 일별, 주별 웹사이트 사용 시간
  • 사용자 설정: 시간 제한 설정, 테마 설정 등

2. 정보 사용 목적

수집된 정보는 다음 목적으로만 사용됩니다:

  • 웹사이트별 사용 시간 추적 및 표시
  • 사용자가 설정한 시간 제한 알림 제공
  • 생산성 분석 및 인사이트 제공
  • 사용자 경험 개선

3. 데이터 저장 및 보안

3.1 로컬 저장

  • 모든 데이터는 Chrome의 로컬 스토리지에만 저장됩니다
  • 외부 서버로 데이터를 전송하지 않습니다
  • 다른 웹사이트나 확장 프로그램이 접근할 수 없습니다

3.2 데이터 보안

  • Chrome의 보안 샌드박스 내에서 작동합니다
  • HTTPS 프로토콜을 사용하는 웹사이트에서만 작동합니다
  • 사용자의 비밀번호나 개인 식별 정보를 수집하지 않습니다

4. 데이터 공유

Tab Timer는 사용자의 데이터를 제3자와 공유하지 않습니다.

  • 광고 네트워크와 데이터를 공유하지 않습니다
  • 분석 서비스에 데이터를 전송하지 않습니다
  • 다른 사용자와 데이터를 공유하지 않습니다

5. 사용자 권리

사용자는 다음과 같은 권리를 가집니다:

  • 데이터 확인: 언제든지 수집된 데이터를 확인할 수 있습니다
  • 데이터 내보내기: JSON 형식으로 전체 데이터를 내보낼 수 있습니다
  • 데이터 삭제: Chrome 설정에서 확장 프로그램 데이터를 완전히 삭제할 수 있습니다
  • 수집 중단: 확장 프로그램을 비활성화하거나 제거하여 데이터 수집을 중단할 수 있습니다

6. 데이터 보존

  • 데이터는 최대 30일간 보존됩니다
  • 30일이 지난 데이터는 자동으로 삭제됩니다
  • 사용자가 직접 데이터를 삭제할 수 있습니다

7. 아동 개인정보 보호

Tab Timer는 13세 미만 아동의 개인정보를 의도적으로 수집하지 않습니다.

8. 개인정보 처리방침 변경

이 개인정보 처리방침은 필요에 따라 업데이트될 수 있습니다. 중요한 변경사항이 있을 경우 확장 프로그램 업데이트를 통해 알려드립니다.

9. 오픈소스

Tab Timer는 오픈소스 프로젝트입니다. 소스 코드는 GitHub에서 확인할 수 있습니다.
