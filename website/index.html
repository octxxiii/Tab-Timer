Tab Timer

웹 사이트 사용 시간 관리 확장 프로그램

소개

Tab Timer는 웹 브라우저에서 각 사이트별 사용 시간을 추적하고 관리할 수 있는 Chrome 확장 프로그램입니다. 생산성 향상과 디지털 웰빙을 위해 웹사이트 사용 시간을 모니터링하고 제한할 수 있습니다.

주요 기능

  • 🕒 실시간 사용 시간 추적
  • ⏰ 사이트별 시간 제한 설정
  • 🔔 시간 제한 도달 시 알림
  • 📊 일일 사용 통계 확인
  • 🌐 모든 웹사이트 지원

설치 방법

  1. Chrome 웹 스토어에서 "Tab Timer" 검색
  2. "Chrome에 추가" 버튼 클릭
  3. 설치 완료 후 Chrome 툴바에서 Tab Timer 아이콘 확인

사용 방법

시간 추적

  • 자동으로 모든 웹사이트의 사용 시간이 추적됩니다
  • 팝업 창에서 실시간 사용 시간 확인 가능

시간 제한 설정

  • 특정 사이트 방문 시 "Set Time Limit" 클릭
  • 원하는 시간(분) 입력
  • "Set" 버튼 클릭

알림 기능

  • 시간 제한 도달 시 자동 알림
  • 알림음 및 시각적 효과 제공
  • "5분 더 사용" 또는 "닫기" 선택 가능
  • 탭을 닫으면 관련 알림 자동 제거

개인정보 보호

  • 모든 데이터는 사용자의 로컬 기기에만 저장됩니다
  • 외부 서버로 데이터 전송이 없습니다
  • 오프라인에서도 정상 작동합니다

기술 스택

  • Chrome Extension API
  • JavaScript (ES6+)
  • HTML5 & CSS3
  • Local Storage
