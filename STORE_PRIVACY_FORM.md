# Chrome Web Store - 개인정보 보호 양식 작성 가이드

## 1. 전용 목적 설명 (1000자 이내)

```
Tab Timer는 사용자가 웹사이트에서 보낸 시간을 자동으로 추적하고 분석하는 시간 관리 확장 프로그램입니다. 사용자는 각 웹사이트별 사용 시간을 확인하고, 시간 제한을 설정하여 디지털 웰빙을 향상시킬 수 있습니다. 모든 데이터는 사용자의 기기에만 로컬로 저장되며, 외부 서버로 전송되지 않습니다. 이 확장 프로그램의 유일한 목적은 사용자가 자신의 웹 사용 습관을 이해하고 개선할 수 있도록 도와주는 것입니다.
```

## 2. 권한 요청 이유

### tabs 사용 근거 (1000자 이내)

```
tabs 권한은 현재 활성 탭의 URL을 확인하여 웹사이트별 사용 시간을 정확하게 추적하기 위해 필요합니다. 사용자가 탭을 전환할 때마다 이전 탭의 도메인과 새 탭의 도메인을 확인하여 시간 추적을 시작하거나 중지합니다. 전체 URL이 아닌 도메인 이름만 추출하여 사용하며(예: google.com), 쿼리 파라미터, 경로, 개인정보가 포함된 URL 부분은 수집하지 않습니다. 이 권한 없이는 웹사이트 사용 시간을 추적할 수 없으므로 확장 프로그램의 핵심 기능을 수행하는 데 필수적입니다.
```

### storage 사용 근거 (1000자 이내)

```
storage 권한은 사용자의 웹사이트별 사용 시간 데이터, 세션 정보, 시간 제한 설정, 사용자 설정(테마, 언어 등)을 사용자의 기기에 로컬로 저장하기 위해 필요합니다. Chrome의 chrome.storage.local API를 사용하여 모든 데이터를 사용자의 브라우저 내부에만 저장하며, 외부 서버나 클라우드 저장소로 전송하지 않습니다. 이 권한은 사용자가 확장 프로그램을 닫았다가 다시 열어도 데이터가 유지되도록 하며, 오프라인에서도 정상 작동하도록 합니다. 데이터는 사용자가 명시적으로 삭제하거나 확장 프로그램을 제거할 때까지 보관됩니다.
```

### activeTab 사용 근거 (필요시 추가)

```
activeTab 권한은 사용자가 확장 프로그램 아이콘을 클릭했을 때 현재 활성 탭의 도메인 정보를 확인하기 위해 필요합니다. 이 권한은 사용자가 명시적으로 확장 프로그램과 상호작용할 때만 활성화되며, 모든 탭에 대한 지속적인 접근을 제공하지 않습니다. 현재 탭의 도메인을 확인하여 시간 제한 설정 및 실시간 시간 추적 기능을 제공합니다.
```

### notifications 사용 근거 (필요시 추가)

```
notifications 권한은 사용자가 설정한 시간 제한에 도달했을 때 알림을 표시하기 위해 필요합니다. 사용자가 특정 웹사이트에 시간 제한을 설정한 경우, 제한 시간에 도달하면 브라우저 알림을 통해 사용자에게 알립니다. 이는 사용자가 자신의 웹 사용 습관을 개선하고 디지털 웰빙을 향상시키는 데 도움이 됩니다.
```

## 3. 원격 코드 사용

**선택**: 아니요, 원격 코드 권한을 사용하고 있지 않습니다.

## 4. 사용자 데이터 사용

### 수집하는 사용자 데이터

다음 항목을 체크하세요:

- [ ] 개인 식별 정보
- [ ] 건강 정보
- [ ] 금융 및 결제 정보
- [ ] 인증 정보
- [ ] 개인적인 커뮤니케이션
- [ ] 위치
- [x] **웹 기록** - 사용자가 방문한 웹사이트의 도메인 이름과 방문 시각 (전체 URL이 아닌 도메인만)
- [x] **사용자 활동** - 각 웹사이트에서 보낸 시간 (분 단위)
- [ ] 웹사이트 콘텐츠

### 데이터 수집 상세 설명

**웹 기록**:
- 수집 항목: 방문한 웹사이트의 도메인 이름 (예: google.com, github.com)
- 수집하지 않는 항목: 전체 URL, 쿼리 파라미터, 경로, 검색어, 페이지 제목
- 목적: 웹사이트별 사용 시간 추적 및 통계 생성
- 저장 위치: 사용자 기기의 Chrome 로컬 저장소

**사용자 활동**:
- 수집 항목: 각 웹사이트에서 보낸 시간 (분 단위), 세션 시작/종료 시간
- 목적: 시간 추적, 통계 생성, 시간 제한 기능
- 저장 위치: 사용자 기기의 Chrome 로컬 저장소

### 확인 사항

다음 세 가지 항목을 모두 확인해야 합니다:

- [x] 승인된 사용 사례를 제외하고 사용자 데이터를 제3자에 판매 또는 전송하지 않음
- [x] 항목의 전용 목적과 관련 없는 목적으로 사용자 데이터를 사용하거나 전송하지 않습니다
- [x] 신용도 판단 또는 대출을 위해 사용자 데이터를 사용하거나 전송하지 않습니다
