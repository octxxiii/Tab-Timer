# 테스트 가이드

## 빠른 테스트 방법

### 1. 빌드 확인

```bash
npm run build
```

빌드가 성공하면 `dist/` 디렉터리에 확장 프로그램 파일이 생성됩니다.

### 2. Chrome에 로드

1. Chrome 브라우저에서 `chrome://extensions/` 접속
2. 우측 상단 "개발자 모드" 활성화
3. "압축해제된 확장 프로그램을 로드합니다" 클릭
4. `dist/` 디렉터리 선택

### 3. 기본 기능 테스트

#### 시간 추적 테스트
1. 여러 웹사이트 방문 (예: google.com, github.com, youtube.com)
2. 팝업 열기 (확장 프로그램 아이콘 클릭)
3. 통계 확인:
   - 오늘 총 사용 시간
   - 세션 수
   - 방문한 사이트 수
   - 사이트별 사용 시간

#### 시간 제한 테스트
1. 팝업에서 시간 제한 설정
2. 설정한 시간(분) 입력
3. "설정" 버튼 클릭
4. 시간 초과 시 알림 확인

#### 테마 테스트
1. 팝업에서 테마 토글 버튼 클릭
2. 다크 모드/라이트 모드 전환 확인

### 4. 문제 해결

#### 확장 프로그램이 로드되지 않을 때
- `dist/` 디렉터리가 올바른지 확인
- `manifest.json` 파일이 있는지 확인
- Chrome 콘솔에서 오류 확인 (`chrome://extensions/` → 확장 프로그램 → 오류 보기)

#### 시간이 추적되지 않을 때
- Background script가 실행 중인지 확인
- Chrome 개발자 도구 → Service Worker 확인
- 콘솔에서 오류 메시지 확인

#### 데이터가 표시되지 않을 때
- Chrome Storage 확인: `chrome://extensions/` → Storage 탭
- 팝업 새로고침 (닫았다가 다시 열기)

### 5. 디버깅

#### Background Script 디버깅
1. `chrome://extensions/` 접속
2. Tab Timer 확장 프로그램 찾기
3. "service worker" 클릭
4. 콘솔에서 로그 확인

#### Popup 디버깅
1. 팝업 열기
2. 우클릭 → "검사"
3. 개발자 도구에서 디버깅

### 6. 테스트 체크리스트

- [ ] 확장 프로그램이 정상적으로 로드됨
- [ ] 팝업이 정상적으로 열림
- [ ] 시간 추적이 작동함
- [ ] 통계가 정확하게 표시됨
- [ ] 시간 제한 설정이 작동함
- [ ] 알림이 정상적으로 표시됨
- [ ] 다크 모드 전환이 작동함
- [ ] 데이터가 저장되고 유지됨
- [ ] 여러 탭 전환 시 시간이 정확히 추적됨
- [ ] 브라우저 재시작 후 데이터가 유지됨
